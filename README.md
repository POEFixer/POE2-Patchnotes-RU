## Как читать

Каждый патч живёт в `patches/<версия>/`:
- `ru/` — переведённые секции (по одному файлу на крупный раздел: skills, items, bug fixes и т.д.)
- `original/` — оригинальный английский текст (бекап + база для проверки)
- `meta.json` — версия, дата, source URL, дата перевода
- `README.md` — оглавление переводов для этой версии

## Правила перевода

## Часто задаваемые вопросы

**Почему названия скиллов и предметов на английском?**
Игроки и сайты вроде poe.ninja, poe2db, poe2scout оперируют английскими названиями. Перевод имён собственных создал бы путаницу при поиске билдов и обсуждениях.

**Почему `30%` без пробела?**
Соответствует исходнику и тому, как числа отображаются в игре. См. [STYLE.md §3](./STYLE.md).

**Откуда берётся английский текст?**
Из официальных публикаций GGG на pathofexile.com. Source URL для каждого патча указан в `meta.json`.
