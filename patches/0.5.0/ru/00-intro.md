# Обновление контента 0.5.0 — Path of Exile 2: Return of the Ancients

Расширение Return of the Ancients включает в себя новую лигу, переработку эндгейма Path of Exile 2 с шестью новыми эндгейм-сюжетами, 2 новыми Ascendancy-классами, а также огромное количество другого контента.
Подробнее об этом расширении читайте на [pathofexile2.com/ancients](http://pathofexile2.com/ancients)
