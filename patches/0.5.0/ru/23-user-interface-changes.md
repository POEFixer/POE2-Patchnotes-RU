## Изменения пользовательского интерфейса

- Добавлено: настройки доступности для увеличения размера всплывающих окон. Их можно найти в разделе UI панели опций.
- Добавлено: отдельная клавиша для выполнения dodge roll, которая не заставляет игрока спринтовать при удержании. Эта клавиша по умолчанию не назначена и должна быть назначена вручную в настройках ввода, если вы хотите ею пользоваться.
- Добавлено: опция "Show Only Available" в списке Hideout Decorations при использовании мыши и клавиатуры.
- Для игроков, использующих click to move: нажатие для перемещения во время каналирования скилла, который можно использовать при движении, больше не будет прерывать каналирование. Это должно значительно улучшить удобство использования таких скиллов, как Flame Breath и Incinerate при click to move.
- Улучшены визуальная чёткость, время загрузки и производительность World Maps Актов 1-3. Дополнительно они были изменены, чтобы более точно отражать игровую направленность планировок областей.
- Всплывающие окна с информацией о скиллах больше не перемещают изменённые статы в верх панели. Они по-прежнему показаны синим цветом.
- Уведомление, появляющееся при продаже предмета в Merchant Tab Shop, теперь отображает другое сообщение, когда проданный предмет находится в другой лиге.
- Кнопка reroll в Ritual UI теперь пульсирует, когда у вас есть бесплатные reroll.
- Empowered Minions (от различных модификаторов или источников) теперь обновляют текст "Minion", отображаемый на монстре.
- Augments Stash Tab обновлён и теперь имеет пять секций.
- Marking противника теперь показывает Mark на его полоске жизни.
- Maximum rage, rage loss delay и rage loss per second теперь отображаются в панели персонажа.
- Увеличен максимальный лимит символов в полях поиска до 250 символов.
- Копирование текста предмета теперь всегда копирует формат расширенного описания.
- Поиск по тексту предмета теперь будет соответствовать формату расширенного описания.
- Теперь вы можете задавать Favourite trades в Currency Exchange при использовании мыши и клавиатуры.
- Теперь вы можете закрепить tracking scroll в своём социальном профиле, чтобы отображать его друзьям, гильдии и членам пати.
- Теперь вы можете предпросматривать скины оружия из микротранзакций в магазине косметики без необходимости иметь экипированный соответствующий тип оружия.
- Добавлено: вид камеры сверху для редактирования вашего hideout при использовании контроллера.
- Добавлено: функция быстрой смены валюты, которую вы в данный момент применяете при игре с контроллером.
- Переработан UI Currency Exchange при игре с контроллером
- Улучшен выбор цели для предметов и интерактивных объектов при использовании контроллера.
- Обновлено и реструктурировано меню Options.
- Включена опция внутриигровых часов для PlayStation и Xbox.
