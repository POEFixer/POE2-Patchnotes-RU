- Включены следующие существующие микротранзакции Back Attachment для использования в Path of Exile 2: Apollyon Cloak, Apollyon Wings, Arcane Cloak, Assassin Cloak, Azurite Back Attachment, Balefire Back Attachment, Corsair Back Attachment, Corsair Cloak, Crucible Back Attachment, Darkwood Cloak, Dreadspire Cloak, Empyrean Back Attachment, Glimmersprite Wings, Gorgon Back Attachment, Illusionist Back Attachment, Innocence Cloak, Innocence Wings, Lightbringer Cloak, Lightbringer Wings, Midnight Pact Cloak, Midnight Pact Wings, Misery Swords Back Attachment, Misery Wings, Myrmidon Cloak, Necrotic Cloak, Necrotic Wings, Nightfall Cloak, Nightfall Wings, Orion Back Attachment, Overseer Cloak, Overseer Wings, Pitfighter Back Attachment, Rapture Back Attachment, Scavenger Back Attachment, Seraph Banner Back Attachment, Sin Cloak, Sin Wings, Spectre Cloak, Spectre Wings, Spider Back Attachment, Scavenger Tail Attachment, Starfall Cloak, Starfall Wings, Suffering Back Attachment, Suffering Cloak, Sunprism Cloak, Sunprism Wings, Sunrise Cloak, Sunrise Wings, Twilight Cloak, Vaal Orb Cloak, Vampiric Wings, Vanguard Banner Back Attachment и Verdant Cloak.
- Включены следующие существующие микротранзакции Helmet для использования в Path of Exile 2: Assassin Hood, Black and White Hood, Cursed Hat, Eyrie Hood, Mageguard Hood, Scavenger Hood и Wild Cougar Helmet.
- Включены все оставшиеся микротранзакции Portrait Frame для использования в Path of Exile 2.
- Включены существующие микротранзакции Character Effect Originator's и Awakener's для использования в Path of Exile 2.
- Включены микротранзакции Chest Extra Effect Something Fishy, Trapped Wisps и Not the Bees! для использования в Path of Exile 2.
- Включена существующая микротранзакция Rare Finisher Effect Anchor Drop для использования в Path of Exile 2.
- Включена существующая микротранзакция Torture Mace для использования в Path of Exile 2.
- Включена существующая микротранзакция Mirage Gauntlet для использования в Path of Exile 2.
- Включена существующая микротранзакция Twilight Helmet Attachment для использования в Path of Exile 2.
- Включены существующие микротранзакции Coliseum Helmet Attachment, Swan Crest Helmet Attachment и Aquila Crest Helmet Attachment для использования в Path of Exile 2.
- Скины персонажей Bronze и Silver Marionette теперь имеют второй тир, который позволяет скрыть базовую броню персонажа.
- Exile's Pilfering Ring и Tracking Scroll of Crafting теперь дополнительно отслеживают предметы Hinekora's Lock и Fracturing Orb, полезные для крафтовых результатов высокой ценности.
- Tracking Scroll of Headhunting обновлён, чтобы также отслеживать некоторых новых боссов.
